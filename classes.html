As a private client you will receive: -

  • Much more individual analysis.
  • Advice on how to address postural weaknesses.
  • A developing programme specifically tailored to your needs.
  • Supporting documentation to use at home.

Each hour-long session will start with a warm up and some basic pilates moves.

This allows me to analyse the way you move and refresh postural awareness.

This analysis allows me to develop a tailored programme, which will be followed and built upon in each session.

We can chart your progress by: -

  • Using photos.
  • Completing a weekly diary, which will also help to track lifestyle features which affet your posture and fitness levels.
